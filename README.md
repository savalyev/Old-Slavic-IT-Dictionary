# 📜 Old Slavic IT Dictionary

Словарь, в котором современные IT-термины обретают древнеславянскую душу.
Здесь «сервер» становится «хранилищем», «баг» — «нечистью», а «деплой» — «походом на ворогов».

## Зачем?
Потому что программирование — это древнее искусство, и язык должен это отражать.

## Как устроено
Каждая запись содержит:
- 🖥️ Современный термин
- 📖 Древнеславянский аналог
- 📜 Краткое толкование в духе летописи
